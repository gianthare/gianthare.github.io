    Step 6. CNC Router

    December 22, 2017 by Sungeun

    CNC router (Or Computer Numerical Control router) is a computer-controlled cutting machine related to the hand held router used for cutting various hard materials, such as wood, composites, aluminium, steel, plastics, and foams. (Wikipedia)

    RPM: revolutions per minute (RPM)
    Plunge: 낙하 jump or push quickly and energetically –drill bits
    Laterally: 측면으로 ---end mills
    Ramp 경사면 –end mill할 때 plunge할 때 ramp주기 (나무 결이 다르거나 딱딱한 옹이의 경우 end mill이 깨질수있음)

    Feed rate & Spindle Speed(RPM)
    Chip load chart is not accurate, more like ambiguous as if GPS
    수학, 소리, 온도로 잘 작동하는지 확인할 수 있음.
    수학: step over/down, RPM, Feed rate
    소리: screaming(배고플 때), 우우웅(너무 많이 먹었을 때)
    온도: 멈추고 end mill 만져서 너무 뜨거우면 문제있음

    Vcarve Pro: dxf파일 import해서 2400*1200/thickness/origin
    Select all open vectors> 두 라인 잡고 오른버튼>join >with a line
    Duplicate vetors 체크 없으면 ok
    Toolpath : profile(incut, outcut, oncut)
    순서는 pocket> incut> outcut

    1. pocket
    -Cutting depths( 0에서 0.6mm)
    -Tool select(end mill선택), edit 선택하여 Notes확인 하고 pass depth(diameter의 half값, step over 50%, Spindle speed 12000, Feed rate 180?(upcut은 20%빨리 해도 됨, downcut 40%빼주기, straigtcut은 20빼주기)
    -clear pocket(offset: 안-밖에서 돌면서, raster(지그재그)//동그란거 깎으면 offset이 더 나을 수도/ ramp 꼭 넣기/
    -이름설정
    -3d view에서 확인하고 다시 돌아가서

    2. Incut
    -Cutting depth plywood 두께에서 0.3추가
    -inside cut/ add ramp to toolpath (zig zag)
    -distance 30~40mm(멀수록 부드럽게)/ name holes쓰기
    -incut하는 부분이 insert해서 끼려고하는 부분이면 fillet>dog bone으로 fillet해준다음에 컷하기

    3. outcut
    -endmill 2개 같은 사이즈 미리 썻으면 자동으로 설정됨/ Tabs설정!! (없으면 나무가 움직이면서 망함) add tabs해서 클릭하여 설정 (길수록 많이 필요, 어려운데(코너,샌딩안하고싶은데)는 넣지 말기)
    -Toolpath list 클릭하고 preview visible toolpasth하기
    -Save toolpath하고 저장
    -Copyand paste해서 cnc방에잇는 컴터에

    [CNC작동방법]
    *Shot bot>Tool>Sprindle Speed 꼭 열고 시작해야함! (sprindle이 자동으로 RPM계산위해서)
    1. 기계 emergency버튼 누르고 오른쪽으로 돌리기, reset버튼 누르기
    2. shot bot프로그램 열고 노란버튼 눌러서 origin설정
    방향키 조정(X,Y), Page up/down 조정(Z)
    -X.Y가 Zero가 아니면 x,y클릭해서 zero설정해주기
    -Z 수동설정: Plate이랑 Pincer을 각각 바닥과 탭에 고정시킨 후, 노란색 창끄고 빨간색 창의 Z클릭한다. 이때 Emergency button을 누를 준비하면서 조심히 관찰. Z축이 위 아래로 움직이면서 조정

    *나사 박은 데 원점에 사각형 그리고 시작하기 (3mm) : 자를 물체가 사각형에 닿지 않도록 옮기고 recaculate하기

    *end mil 교체할때 항상 노란색 프로그램 끄기

    *중간에 stop하고 end mill너무 뜨거우면 vcarve다시 열어서 sprindle/feed 다시 조절 하고 시작
