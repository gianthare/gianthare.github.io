    • Step 1. HTML & CSS

    November 28, 2017 by Sungeun

    The first step is to make my own homepage with HTML and CSS on which I log what I study and how I develop as a maker during this training.

    First of all, I had to find a channel to teach me what the heck HTML and CSS is. Fortunately, T-Academy run by SK telecom provides free online courses which I found very useful to capture a basic knowledge in a short period of time. After taking the course, I referred to Bootstrap which has a lot of sample codes for necessary components. I learned the codes I needed for my own homepage layout and adjusted them.

    *Fabrication Laboratory started from MIT course bits and atoms which aims to transform digital bits to physical atoms. In other words, tangible products can be realized using Digital Fabrication and Physical Computing tools such as 3D printers or PCBs. However, these so-called technologies will not replace all analogue tools as modern kitchen has various devices for cooking. Digital and analogue will be used together. It should be noted that how to model and code (i.e. creating contents) is more important than how to use machines like 3D printers.
